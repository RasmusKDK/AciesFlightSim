# Acies Flight Simulator ✈️

Dette projekt er et simpelt flight simulation API bygget i ASP.NET Core.

## Funktioner
- Opret og hent fly og lufthavne
- Simuler flyveture
- Udregn afstand, tid og brændstof

## Teknologier
- ASP.NET Core
- In-memory storage
- Swagger UI

## Sådan kommer du i gang
1. Clone repoet
2. Kør projektet
3. Åben Swagger på `https://localhost:xxxx/swagger`

## Eksempel på simulation
```json
{
  "aircraftId": "b1",
  "fromAirportId": "a1",
